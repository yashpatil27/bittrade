Now create a websocket broadcast service, that broadcasts globally. it will be called btc_price_update. It will be triggered when bitcoin_data.btc_usd_price changes. It should return [btc_usd_price, buy_rate_inr, sell_rate_inr]. These will be calculated as [bitcoin_data.btc_usd_price, bitcoin_data.btc_usd_price * settings.buy_multiplier, bitcoin_data.btc_usd_price * settings.sell_multiplier]
